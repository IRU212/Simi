### 作品概要

- #### 学生を対象とした学習サイト(e-learning)
- #### コミュニケーションが苦手な生徒が友達・教師などに聞きづらい人でもネットを通じて質問が出来る
- #### 生徒が質問した内容を一覧で見て知識の取得を図る
- #### 学習時間や内容を記録する事で学習スケジュールを立てやすくする
